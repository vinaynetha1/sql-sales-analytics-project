# SQL Sales Analytics Project

This project performs end-to-end sales analysis using SQL.

## Objectives

- Analyze sales trends over time
- Calculate cumulative sales growth
- Evaluate product performance
- Measure category contribution to revenue
- Segment customers and products
- Build customer and product reporting views

## SQL Techniques Used

- Window Functions (SUM OVER, AVG OVER, LAG)
- Common Table Expressions (CTE)
- Aggregations
- CASE statements
- Data segmentation
- Analytical views
